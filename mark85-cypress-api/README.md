## Configure os acessos no arquivo .env

## Instalação

Instalar as dependências: API, MAIL e WEB

```bash
  npm install
```

## Execute os testes do cypress via terminal

```bash
  npm run test
```

## Execute os testes do cypress via navegador

```bash
  npm run open
```
